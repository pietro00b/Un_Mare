# Brano Un_mare

Progetto Pure Data per performance live.

## Struttura

```
pure_data/
├── Main.pd          # Patch principale
├── sezione1.pd      # Sezione 1
├── sezione2.pd      # Sezione 2
├── sezione3.pd      # Sezione 3

```

## Requisiti

- [Pure Data](https://puredata.info/) (Pd-vanilla o Pd-extended)

## Utilizzo

Aprire `src/Main.pd` con Pure Data per avviare la patch principale.
